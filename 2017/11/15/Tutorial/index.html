指令

init

新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

new

新建一篇文章。如果没有设置 layout 的话，默认使用_config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。
